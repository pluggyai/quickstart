# Pluggy Connect - HTML Example

Minimal integration of the Pluggy Connect widget using vanilla HTML and JavaScript. No build tools required.

## How It Works

1. Loads `pluggy-connect.js` from the Pluggy CDN
2. Fetches a connect token from your backend API
3. Initializes the Pluggy Connect widget

## Setup

1. Open `index.html`
2. Replace `'<your url to get the connect token here>'` with your token endpoint URL
   - You can deploy one in minutes using the [Vercel Connect Token example](../../examples/vercel-node-connect-token)
3. Open `index.html` in a browser

No `npm install` or build step needed.
